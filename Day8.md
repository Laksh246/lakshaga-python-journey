# Day 8 - Functions

## What I learned
Functions are reusable blocks of code that perform a task.

---

## Real life example: Coffee machine
Press one button → coffee is made automatically.

---

## Mobile Photo Capture Example
Photo button perform actions like capture images or record videos.

---

You open your phone camera and press:
“Take Photo” button

WHAT ACTUALLY HAPPENS

You don’t do steps manually like:
open lens
adjust focus
capture image
save file
The phone does EVERYTHING automatically.
⚙️ THIS IS A FUNCTION

One button = one function
Many hidden steps inside

## Python idea
```python
def take_photo():
    print("Opening camera")
    print("Focusing lens")
    print("Capturing image")
    print("Saving photo")
```
---
```python
take_photo()
```
## My thought
Now I understand that functions help avoid repeating the same code again and again. A function is like a mobile app button that hides complex steps and gives one result.
