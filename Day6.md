## Example

## Real Life Example: Brushing Teeth

We don’t brush teeth only once.

We repeat it every day as a habit.

### Thinking like a loop:
- Day 1 → Brush teeth  
- Day 2 → Brush teeth  
- Day 3 → Brush teeth  
- ... continues every day  

This repetition is called a loop.

## Python idea
Loops allow repetition of tasks automatically.
```python
for day in range(7):
    print("Brush teeth")
```
