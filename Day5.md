# Day 5 - IF / ELSE (Decision Making)

## What I learned
Programs can make decisions using if and else.

## ATM example
If PIN is correct → allow access  
Else → deny access

## Phone example
If fingerprint matches → unlock  
Else → reject

## Python idea
if condition:
    do something  
else:
    do something else
