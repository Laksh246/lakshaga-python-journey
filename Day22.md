# Day 22 - From Information to Knowledge (Making Decisions from Information)

## What I Learned

Data is raw facts.

Information is data that answers a question.

Knowledge is understanding gained from information that helps make decisions.

A computer can store data and find information.

Humans use information to create knowledge and make decisions.

---

## Learning Journey So Far

```text
Data
  ↓
Information
  ↓

```

---

## Understanding the Difference

### Data

Raw facts.

Example:

```text
Book Title → Python Basics
Price      → ₹500
```

These are simply stored values.

---

### Information

Data that answers a question.

Question:

```text
Which books cost less than ₹600?
```

Answer:

```text
Python Basics → ₹500
```

Now the data has become information.

---

### Knowledge

Understanding gained from information.

Example:

Information:

```text
Python Basics → ₹500
Data Science  → ₹700
AI Fundamentals → ₹900
```

Knowledge:

```text
Python Basics is the cheapest book.
```

This conclusion is knowledge.
